🔧 Server Prerequisites

Before running the Python server, ensure the following prerequisites are met:

📁 Directory Structure

  • /html – contains all HTML pages
  • /scripts – contains JavaScript files

🐍 Python Requirements

  • Python 3.6 or higher
  • Standard library modules used:
    • http.server
    • socketserver
    • os
    • json
    • http.client
    • ssl
  • No external packages required

🚀 Setup Instructions

python your_server_script.py

Once running, visit http://localhost:8090 in your browser.

🌐 Network Requirements

  • External API server must be reachable at https://192.168.8.150:4443
  • Port 8090 must be open and not blocked by firewall

⚠️ Notes

  • SSL verification is disabled for development purposes using ssl._create_unverified_context()
  • Ensure JSON payloads sent to /home and /success are properly formatted
